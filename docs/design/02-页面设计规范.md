## 版本更新记录

### v1.2 (2026-02-09)
- **新增**: 主题切换器交互规范
  - Hover 展开模式：鼠标悬停显示所有主题列表
  - 实时预览：Hover 主题行时页面颜色实时变化

  - 移除重复的"数据分析"标题（DashboardLayout 已包含）
  - 热门表单排行更名为"热门TOP10表单排行"
  - 保持两行统计卡片布局

### v1.1 (2026-02-08)
- **更新**: 数据分析页面按钮样式
  - 刷新数据按钮替代分享结果按钮

## 1. 页面架构总览

```
MultiForms 网站架构
│
├── Landing Page (首页)           # 营销页面，产品介绍
├── Auth Pages (认证页面)
│   ├── Login                     # 登录
│   ├── Register                  # 注册
│   └── Forgot Password           # 忘记密码
├── App Pages (应用页面)
│   ├── Dashboard                 # 仪表盘
│   ├── Form Builder              # 表单构建器
│   ├── Form List                 # 表单列表
│   ├── Form Analytics            # 数据分析
│   ├── Settings                  # 设置
│   └── Profile                   # 个人资料
└── Public Pages (公开页面)
    ├── Form View                 # 表单填写页
    ├── Form Result               # 结果展示页
    └── Share                     # 分享页面
```

### 12.3 主题变量系统

所有主题通过 CSS 变量定义，核心变量包括：

```css
/* 主色调渐变 */
--primary-start: #6366F1;
--primary-end: #8B5CF6;
--primary-glow: #A78BFA;

/* 强调色 */
--accent-color: #EC4899;

/* 背景色 */
--bg-primary: #0F0F23;
--bg-secondary: #1A1A2E;
--bg-tertiary: #2D2D44;

/* 文字色 */
--text-primary: #F8FAFC;
--text-secondary: #94A3B8;

/* 玻璃态效果 */
--glass-bg: rgba(26, 26, 46, 0.6);
--border: rgba(255, 255, 255, 0.08);
```

### 12.4 主题应用范围

主题色应用到以下组件：
- 按钮（主按钮、边框颜色）
- 输入框（focus 状态）
- 图表（渐变填充）
- 进度条
- 徽章和标签
- 链接 hover 状态
