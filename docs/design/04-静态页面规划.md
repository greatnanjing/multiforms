## 1. 页面架构总览

```
MultiForms 静态页面结构
│
├── 01. Landing Page (首页)
│   ├── Hero Section
│   ├── Features Section
│   ├── Templates Showcase
│   ├── Testimonials Section
│   └── CTA Section
│
├── 02. Auth Pages (认证页面)
│   ├── Login (登录) - 前台用户登录
│   ├── Register (注册)
│   ├── Forgot Password (忘记密码)
│   └── Admin Login (管理员登录) - 独立的管理后台登录入口
│
├── 03. App Pages (前台应用页面 - 表单创建者)
│   ├── Dashboard (仪表盘)
│   ├── Form Builder (表单构建器)
│   ├── Form List (表单列表)
│   ├── Form Analytics (数据分析)
│   ├── Form Settings (表单设置)
│   └── Profile (个人资料)
│
├── 04. Admin Pages (管理后台页面 - 管理员)
│   ├── Admin Dashboard (管理员仪表盘)
│   ├── User Management (用户管理)
│   ├── Form Management (表单管理)
│   ├── Content Review (内容审核)
│   ├── Template Management (模板管理)
│   ├── System Settings (系统设置)
│   └── System Logs (系统日志)
│
├── 05. Public Pages (公开页面 - 所有访客)
│   ├── Form View (表单填写页)
│   ├── Form Result (结果展示页)
│   └── Password Gate (密码验证页)
│
├── 06. Templates Pages (模板库 - 前台用户)
│   ├── Templates List (模板列表)
│   └── Template Detail (模板预览)
│
└── 07. Utility Pages (工具页面)
    ├── 404 Page
    ├── 500 Server Error
    └── Coming Soon
```

### 1.1 路由架构

| 路由前缀 | 说明 | 访问权限 |
|----------|------|----------|
| `/` | 公开页面 | 所有人 |
| `/login`, `/register` | 前台认证 | 未登录用户 |
| `/dashboard` | 前台应用（仪表盘） | 表单创建者 |
| `/forms/*` | 表单管理相关页面 | 表单创建者 |
| `/templates` | 模板库 | 所有人 |
| `/profile` | 个人资料 | 表单创建者 |
| `/f/:shortId` | 公开表单 | 所有人 |
| `/admin` | 管理后台登录 | 管理员 |
| `/admin/*` | 管理后台页面 | 管理员 |

### 15.10 管理后台设计要点

| 方面 | 规范 |
|------|------|
| **配色** | 保持品牌色，但使用更专业的深色侧边栏 |
| **布局** | 固定侧边栏 + 内容区，最大化数据展示空间 |
| **表格** | 紧凑型设计，支持排序、筛选、批量操作 |
| **操作** | 主要操作放在右上角，行内操作用图标按钮 |
| **响应式** | 管理后台主要面向桌面端，平板做基本适配 |
| **权限** | 所有 `/admin/*` 路由需验证管理员权限 |
