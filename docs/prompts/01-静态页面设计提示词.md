# 设计原则

1. **深色主题优先** - 默认使用深色主题
2. **禁用 Emoji 作为图标** - 使用 SVG 内联图标（Lucide/Heroicons 风格）
3. **必须有 Hover 状态** - 所有交互元素需要 hover 反馈
4. **禁止布局抖动** - Hover 效果不应导致布局变化
5. **品牌渐变色** - 使用定义的渐变色，不使用随机颜色
6. **适度玻璃态** - 有节制地使用玻璃态效果
7. **中文为主** - 主要语言为简体中文 (zh-CN)

## 常见问题

### Q: 如何确保页面完全响应式？

A: 至少包含以下媒体查询：
```css
@media (max-width: 1280px) { /* 大屏幕调整 */ }
@media (max-width: 1024px) { /* 平板横屏 */ }
@media (max-width: 768px)  { /* 平板竖屏 */ }
@media (max-width: 640px)  { /* 手机 */ }
@media (max-width: 375px)  { /* 小屏手机 */ }
```

### Q: 如何实现玻璃态效果？

A: 使用以下 CSS：
```css
background: rgba(26, 26, 46, 0.6);
backdrop-filter: blur(20px) saturate(180%);
border: 1px solid rgba(255, 255, 255, 0.08);
```

### Q: 图标从哪里获取？

A: 从 [Lucide](https://lucide.dev/) 或 [Heroicons](https://heroicons.com/) 复制 SVG 代码，内联到 HTML 中。不要使用 emoji 作为图标。
